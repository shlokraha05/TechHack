PARIVESH 3.0

About the Portal

PARIVESH 3.0 is a next-generation, role-based workflow system designed to improve transparency, efficiency, and speed in environmental clearances

The portal facilitates the end-to-end lifecycle of an application—from initial filing by a proponent to the final publication of the Minutes of the Meeting (MoM)

Admin Control

Manages dynamic role assignments and provisions master templates for "Gist" documents based on application categories (A, B1, B2)

Proponent Filing

Provides a multi-step digital form for project details, secure document attachments, and integrated UPI/QR fee payments

Verification Unit

Reviews documents and flags deficiencies via Essential Document Sought (EDS) to return applications for correction

Secretariat Security

Transforms edited gists into formal MoMs and locks them to prevent further edits after publication

Linear Workflow Integrity

Draft Submitted Under Scrutiny Referred Finalized

Applications flow linearly to ensure security and data isolation between roles
